# ecommerce_basic (PHP + MySQL)

## 📌 Project Overview

This is a full-stack **E-Commerce Web Application** built using **PHP, MySQL, HTML, CSS**.
The application allows users to browse products, register/login, add items to cart, and manage their shopping experience.

---

## 🚀 Features

* 👤 User Registration & Login
* 🛍️ Product Listing from Database
* 🛒 Add to Cart Functionality
* 🔄 Update / Remove Cart Items
* 🔐 Secure Password Hashing
* 📦 Session-based User Management

## ⚙️ Setup Instructions

### 1️⃣ Clone the Repository

```
git clone https://github.com/your-username/ecommerce.git
```

### 2️⃣ Move Project to XAMPP

Place the folder inside:

```
C:\xampp\htdocs\
```

---

### 3️⃣ Start Server

* Open XAMPP
* Start:

  * Apache ✅
  * MySQL ✅

### 4️⃣ Create Database

* Open: http://localhost/phpmyadmin
* Create database:
